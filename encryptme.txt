HOW TO SET UP A GITHUB REPOSITORY

1. Copy pathname of project into Terminal
2. Run command ‘git init’ and run ‘git status’ to ensure it’s running properly. Must say following if there are no files
On branch main

No commits yet

Untracked files:
  (use "git add <file>..." to include in what will be committed)
	.idea/
3. Run commands:
git add .
git commit -m "Initial commit”
4. Add src file and be sure to mark it as ‘Sources Root’
5. Done
